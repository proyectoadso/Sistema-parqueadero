Bienvenido al Sistema de Parqueadero

El Sistema de Parqueadero es una herramienta que permite el control de entrada y salida de vehículos (carros, motos y bicicletas) de un parqueadero tarifando tiempos por minutos, horas, bloques de horas, bloques de tiempo, días, quincenas o mensualidades.

Sistema más que intuitivo

Interfaz gráfica agradable, refinada en cada detalle para aprovechar al máximo el espacio de la pantalla, textos de tamaño grande para una mejor legibilidad por parte del usuario y clientes. Un diseño pensado para el menor uso de clics del ratón y pulsaciones del teclado.

Motor de Base de datos

Posee un motor de Base de Datos MySQL 5.7 el cual permite una rápida velocidad de acceso a los datos, gran capacidad de almacenamiento y conectividad segura a través de perfiles de usuario. Copias de seguridad automáticas de forma diaria y en varios momentos del día para una tranquilidad total.
